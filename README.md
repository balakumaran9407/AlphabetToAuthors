# Alphabet to Authors

Alphabet to Authors is an interactive language mastery platform offering General English, Creative Writing, Public Speaking, and Spoken English courses for learners of all ages.

## Features
* **Modern Web Interface:** Fully responsive design built with HTML, CSS, and vanilla JavaScript.
* **Interactive Modals:** Dynamic course syllabus and detail views.
* **Firebase Integration:** Real-time feedback submission using Firebase Firestore.
* **SEO Optimized:** Sitemaps, robots.txt, and Open Graph tags for optimal search engine visibility.

## Live Website
Visit the live website at: [https://alphabettoauthors.web.app](https://alphabettoauthors.web.app)

## Local Development
1. Clone the repository
2. Open `index.html` in your browser, or use a local server like Live Server (VS Code).

## Deployment
This project is configured to deploy to Firebase Hosting. 
To deploy updates:
```bash
firebase deploy --only hosting
```
